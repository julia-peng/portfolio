Traffic Light System

This was a project that I made for the TEJ3M course, with my partners Maya Peski and Elya Keren-Sagiv.
It was meant to introduce us to the process of design, neatly wrapping up all content that had been learned up to that point in the course, including basic skills in C, 3D design software, boolean decisions and hard implementation of systems through circuits and Arduino.
In this project, I was put in charge of the TinkerCAD, which was meant to create a baseline off which the physical model could be created. This meant figuring out the wiring and, most importantly of all, the C code that would drive the real-life system. In this system, there would have to be impelmentations for traffic lights and pedestrian buttons with proper timing, buttons to control servo motors and LDR sensors to turn on/off with a certain amount of light.
Overall, it was an enjoyable project. Though it did require a lot of work and coordination with my group members, it turned out quite well.
