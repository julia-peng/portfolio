Country Sort

The country sort task was an assignment given to me as part of the ICS4U curriculum. I completed this assignment with Lily Liu and Jonathan Ye.
The country sort task revolved around sorting algorithms and storing values. A list of countries was provided to us in a simply formatted text file; however, some of the countries' names made it difficult to correctly separate each individual line of input into the country and capital names.
To combat this issue, what we did was we created a String that held all the countries that required exception handling, as well as their index in the array. Then, if the inputted country matched one of the exceptions found in the String, the program would be equipped to handle it accordingly.
Check out the program at the GitHub page.
