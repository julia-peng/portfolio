Portfolio Website

Another project that I undertook as part of my course work for the ICS4U curriculum was the development of a personal portfolio website. Using the tiny base of knowledge of HTML I had learned from a free online course nearly 4 years ago, I used various online resources to guide my development of this website.
Initially, I was planning on using a website builder to create my wesite. however, I decided to use HTML instead as I thought it would be a project that provided not only valuable learning experience, but could also undergo continued development and form my proper portfolio website later on.
It uses HTML5 and basic CSS to develop the layout and aesthetics of the website.
This is a little ironic.
